This program takes in user data and sends it to a database. 
It then fetches the updated data from the database to display on on the webpage, with some data manipulations.

Front end and backend (called usersystem) folder is available to view code. 
Backend can be executed from just the jar. Java 17 and 
Springboot version 2.7.8 is used with dependencies are included. 

To run program:

1- Download or clone repo.
2- Start MySql server on port 3307
3- Execute backend jar file
4- In front end directory, run: npm start
5. Open http://localhost:3000 to view it in the browser.
